Discrimination gauche / droite

Une image de main va s'afficher, sous différents angles et différentes vues (dos ou paume). Indiquez le plus rapidement et précisément possible s'il s'agit d'une main gauche ou droite, en imaginant mentalement votre propre main dans cette position.

Vous pouvez répondre en cliquant sur les boutons à l'écran, ou avec les flèches ← / → du clavier.

Note pour l'équipe de recherche : ce fichier contient 3 modes (Base, Intermédiaire, Complexe), chacun avec 3 niveaux de difficulté (24 essais chacun) et un modèle de main dédié. Le mode « Complexe » utilise actuellement une agrafeuse comme objet ; deux photos de stylo ont été reçues mais nécessitent une vue de paume manquante pour être ajoutées comme second modèle. Envoyez-moi de nouvelles photos (main droite, doigts vers le haut, fond neutre, dos + paume) pour enrichir n'importe quel mode. Vérifiez l'accord de la personne photographiée avant diffusion.
